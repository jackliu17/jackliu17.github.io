Swift中的字符串和集合

发表于 2016-03-10   |   分类于 IT   |  

字符串

Swift中的字符串是结构体，而NSString中的字符串是Class

表现形式

var s:Character = "美"
var quote1 = "\'"
var quote2 = "\u{22}"
print("quote1的值为:\(quote1),quote2的值为:\(quote2)")
  1. 单个字符来指定字符常量，如“A”，“9”
  2. 转义字符表示特殊字符常量，如”\n”,”\t”
  3. 使用\u{n}的Unicode形式，n代表一个1~8位的十六进制
  4. 必须用双引号包起来
  5. Swift中的每一个字符代表了一个可扩展字母集

字符串创建

方法1：直接赋值

var str1 = "Jike"

方法2：构造函数

var str2 = String()

//创建多个重复字符的字符串
var str3 = String(count:4,repeatedValue:Character("a"))
var strRep = String(count:4,repeatedValue:UnicodeScalar("xxxxx")) //使用unicode
print(strRep)

字符串是否为空

print(str3.isEmpty)

字符串拼接

方式1.用 “+”

var str4 = str1 + "College"
str1 += "Hello"
print(str4)

方式2.用append

let char:Character = "!"
str1.append(char)
print(str1)

字符数量

Swift1.x

print("str3 has \(countElements(str3)) characters")

Swift2.x

print("str3 has \(str4.characters.count) characters")
  1. Swift中的Unicode16Count函数所获得的值，才是和NSString中的length一样。因为SWIFT的count是基于Unicode，而OC的length是基于UTF-16
  2. Swift中的字符串是否可以修改仅通过定义的是变量还是常量来决定,也就是var or let

字符串比较

let quotation = "We are the same"
let sameQuotation = "We are the same"
if quotation == sameQuotation {
print("These are the same")
}

字符串是否有特定的前缀后缀

var food = ["Fruits:apple",
"Fruits : orange",
"Fruits : banana",
"Vegetables: tomato",
"Vegetables: potato"]

for fry in food{
if fry.hasPrefix("Fruits"){
   print(fry)
}

if fry.hasSuffix("o"){
   print(fry)
}
}

字符串转为整形

Swift 1.x

let myString: String = "256"
let myInt: Int? = myString.toInt()

Swift 2.x

let myString: String = "256"
let myInt: Int? = Int(myString)

数组

数组的创建

方式1.

var myArr = Array<String>()  
var num = Array<Int>(count:3,repeatedValue:1)

方式2.

var someInts = [Int]()
var arr:[Int]=[1,2,3]
var threeDoubles = [Double](count:3, repeatedValue:0.0)
var food = ["apple","orange","banana"]
print(food[3])
print(food.count)

AnyObject and Any

Swift中有两种特殊的object:

  1. Any:表示任何数据类型
  2. AnyObject:表示任何对象
for fruit in food{
fruit = "good" //error, fruit默认是let类型
}

数组的可变性

添加

1. 用append()在数组尾部添加

food.append("Vegetables:mushroom")

2. 通过加法添加数组元素

food += ["pineapple","pity"]

替换

food[0...2] = ["a","b"]
//从第一个到第三个元素进行替换，但只有两个元素，第三个将被删除，被“没有”所替换

插入

food.insert("Meat",atIndex:0)

删除某一项

food.removeAtIndex(0)

删除最后一项

food.removeLast()

删除所有

food.removeAll(keepCapacity:false)

字典

声明和创建

字典的key or value 通常最好是同一类型，否则强制解析时会出错
字典的遍历后，返回时元组类型

//Dictionary<KeyType,valueType>
var dic1: Dictionary<String,String>

//[KeyType:valueType]
var scores:[String:Int]

dic1 = Dictionary<String,String>()
scores = Dictionary<String,Int>(minimumCapacity:5)

空字典

var emptyDic:[String:Double] = [:] 
print(emptyDic.isEmpty)

字典的使用

//这种的声明是不严谨的，因为值的类型都不是一样的，系统会推断它是NSObject类型
var person = ["age":18,"name":"Tom","身高":178]

//这样会报错，因为是将NSObject转为int
var height:Int? = person["身高"]  
if height != nil {
   print(height!)
}

//以下才正确
var person = ["age":"18","name":"Tom","身高":"178"]

var height:String? = person["身高"]  
if height != nil {
   print(height!)
}

print(person["age"])

//不存才的key，返回为nil
print(person["weight"])

//对该值重新赋值
person["name"] = "Jack"

//对不存在的key设置value,字典将会添加k-v对
person["weight"] = 90
  1. 数组是有序的，字典是无序的
  2. 通过常量或者变量决定字典是否可变
  3. 如果没有显示数据类型声明，字典就会根据值进行推断

字典的替换

  1. 该方法返回更新值之前的原值
  2. 键不存在对应值的时候设置值
  3. 键存在时更新已存在的值
  4. 键不存在时，不发生替换，不返回任何东西
if let oldName = person.updateValue("Lucky",forKey:"name"){
   print("原来的值是:\(oldName)")
}

字典的遍历

//使用元组
for (key,value) in person{
 print("\(key):\(value)")
}

字典的删除

//移除单个值
person.removeValueForKey("name")

//移除所有值
person.removeAll(keepCapacity:false)

字典键和值的取出

let keysArr = [String](person.keys)
let valueArr = [String](person.values)
print(valuesArr)

总结

  1. 字符串的创建，拼接，字符数量，转整型，是否为空，是否有前后缀
  2. 数组的创建(左右使用方括号),添加，替换,插入，删除
  3. 字典的创建(和数组差不多，就方括号里多个值)
  4. 字典的key or value 通常最好是同一类型，否则强制解析时会出错.如果没有显示数据类型声明，字典就会根据值进行推断因为.系统就会认为是NSObject
  5. 数组是有序的，字典是无序的
