Swift面向对象基础(上)

发表于 2016-03-11   |   分类于 IT   |  

面向对象的特点

  1. 面向对象编程的三大特征：封装，继承，和多态
  2. Swift可以定义枚举，结构体，和类三种面向对象的类型

枚举

枚举定义

  1. 用于管理一组有限的值的集合
  2. 支持计算属性
  3. 支持实例方法和类方法
  4. 支持定义构造器来完成初始化
  5. 支持扩展和协议

枚举语法

enum 枚举名 {
  //使用case关键字列出所有枚举值
  //枚举的其他成员
}

//方法1
enum Season {
   case Spring
   case Summer
   case Fall
   case Winter
}

//方法2
enum Season2 {
   case Spring, Summer, Fall, Winter
}

使用枚举声明变量

var weather: Season
weather = Season.Fall
weather = .Fall //也可以这样写
print(weather)

枚举值和switch语句

switch中的case没有覆盖枚举的所有值，必须添加default语句

var chooseDay = Season.Fall

switch(chooseDay){
case .Spring:
  print("This is spring")
case .Summer:
  print("This is summer")
default:
  print("At home")
}

原始值

enum 枚举名：原始值类型{
   case 枚举值 = 原始值
}

整型的枚举

//可以自动推断，不用每个都赋予原始值
enum Weekday: Int{
  case Mon, Tur, Wen = 3, Thur, Fri, Sat , Sun
}

字符串枚举

//必须每个都赋予原始值，因为不能自动推断
enum Season3: Character{
  case Spring = "春"
  case Summer = "夏"
  case Fall = "秋"
  case Winter = "冬"
}

获取原始值

print(Weekday.Fri.rawValue)

获取枚举值

var mySeason = Season3(rawValue:"春")
if mySeason != nil {
  switch(mySeason!){ //不为空才能进行强制解析
  case .Spring:
    print("春天不是读书天")
  case .Summer:
    print("夏天炎炎正好眠")
  case .Fall , .Winter:
    print("秋多蚊蝇冬日冷")
  default:
    print("读书只好等明年")

  }
}

关联值

enum Planet {
case Earth(weight: Double , name:String)
case Mars (density: Double, name: String, weight:Double)
case Venus(Double, String)
case Saturn
case Neptune
}

var p1 = Planet.Earth(weight:1.0, name:"地球")
var p2 = Planet.Venus(0.815,"金星")
var p3 = Planet.Mars(density:3.95, name:"火星",weight:0.1)

switch (p3){
//将关联值绑定到变量或者常量来提取
case Planet.Earth(var weight , var name):
   print("此行星的名字为:\(name),质量相当于\(weight)个地球")

//将关联值都提取为常量或者变量，只将一个var或者let放在枚举成员之前
case let Planet.Mars(density:d, name:n, weight:w):
   print("此行星的名字为:\(n),质量相当于\(w)个地球，密度为\(d)")
default:
   break
}

类和结构体

  1. 这两个非常相似
  2. 主要区别在于结构体和枚举不支持继承，只有类才有继承
  3. 结构体和枚举是值类型
  4. 类是引用类型
  5. 只有类的实例可以称为对象

定义

类

[修饰符]class类名 {
零到多个构造器
零到多个属性
零到多个方法
零到多个下标
}
修饰符可以是private ,public, internal,final

class Person {
var name: String = "Jack"
var age: Int = 0
func say(content: String)
{
   print(content)
}
}

结构体

[修饰符]struct 结构体名 {

}
修饰符可以是private , public, internal

struct Dog{

var name: String
var age: Int
func run(){

print("\(name)在奔跑")
}
}

属性

[修饰符]var或者let 存储属性:类型名字 = 初始值

构造器

[修饰符]init(形参列表){
//零到多条可执行语句组成的构造器执行体
}

方法

[修饰符]func 方法名(形参列表) ->返回值类型 {
//零到多条可执行语句
}

创建实例

类

var p: Person
p = Person() //等价于 var p = Person()
print(p.name)
p.say("Hello world")

结构体

var dog = Dog(name:"Wangcai", age:2)
print(dog.name)
dog.run()

值类型与引用类型

引用类型

var p2 =p
p2.name = "Rose"
print(p.name)
  1. 内存里的对象可以有多个引用，即多个引用变量指向同一个对象
  2. 如果一个引用更改了，那对象里的值也会更改。

值类型

var dog2 = dog
dog2.name = "snoopy"
print(dog2.name)
print(dog.name)
  1. 值对象是要复制的
  2. 每个值对象的更改，不会影响其他对象的更改
  3. 因此枚举和结构体是没有继承的

引用类型的比较

class User {
var name: String
var age: Int
init(name: String, age: Int){
    self.name = name
    self.age = age
}
}

var u1 = User(name:"Han",age:32)
var u2 = User(name:"Han",age:32)
//虽然传递的值是一样，但是是创建了两个对象

print(u1 === u2) //u1与u2引用的不是同一对象,结果是false
print(u1 !== u2) //结果是true

var u3 = u1
print(u3 === u1) //结果是true

// "===" 这个等号只能用于引用类型的比较，不能用于值类型的比较，所以不能用于枚举或者结构体
// “==” or “!=” ,这两个也可以用于比较引用类型，但是要用到运算符重载，详情见后期课程

self关键字

方法中的self代表该方法的调用者

class Dog {  
  func jump(){
    print("正在执行jump方法")
  }
  func run(){
  self.jump()  //这里的self代表该方法的调用者，这里也可以去掉
     print("正在执行run方法")
  }
}

构造器中的self代表该构造器正在初始化的实例

class Person {
    var name: String = ""
    var age: Int = 2

    init(name:String, age:Int){
    self.name = name
    self.age = age
    }

    func info(){
    print("我的名字是:\(name)年龄是:\(age)")
    }
}

var person = Person(name:"Lily",age:10)
person.info()

类和结构体的选择

  1. 结构体的主要目的是用于封装少量相关的简单数据
  2. 如果需要在传递参数或者赋值时自动复制副本，使用结构体
  3. 明确该类型无须继承另一个已有的类或被其他类继承
  4. 注意，大部分时候，程序应该自定义类而不是自定义结构体。

总结:

  1. Swift中的三种面向对象类型:枚举，结构体，类
  2. 枚举,一组有限的值的集合.要区分枚举值，关联值，原始值
  3. 类和结构体很相似。类是引用类型，支持继承;结构体和枚举是值类型，不支持继承
  4. 值类型和引用类型的区别。
