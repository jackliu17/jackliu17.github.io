如何在xcode里正确的建立文件夹？

  1. 如果仅是在xcode里创建，那文件夹不会在存在finder里，实际上相关文件还是存在一个目录下
  2. 需要在Xcode里左下角，”add new file to ….project” –> create new folder –> add,这样创建的文件夹可以存在finder里
